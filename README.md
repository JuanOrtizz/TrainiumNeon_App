# TrainiumNeon App

Aplicación de gimnasio desarrollada en **.NET MAUI** para gestionar rutinas de entrenamiento, registrar progresos y facilitar el seguimiento de tus objetivos fitness.

## Características principales

- **Login / Registro**  
  - Registro y Autenticación de usuarios con correo y contraseña.  

- **Menú Principal**  
  - Muestra los **ejercicios del día** con la **rutina seleccionada**.  

- **Mi cuenta**  
  - Visualiza y edita tus datos personales (nombre, email, etc.).  

- **Rutinas**  
  - Lista de rutinas creadas por el usuario.  
  - Opciones para **agregar**, **modificar**, **eliminar** y **seleccionar** la rutina que se mostrará en el menú principal.

- **Detalle de Rutina**  
  - Muestra los ejercicios y la cantidad de series de la rutina seleccionada.

- **Estadísticas**  
  - Registra y modifica **PR (Personal Records)**.  
  - Filtra PR por peso máximo/mínimo y por orden alfabético.  
  - Permite eliminar PR si es necesario.


## Tecnologias

- **C#**
- **.NET MAUI**
- **SQLite**
- **Patrón MVVM**

## Ejecución

- Clonar repositorio
- Abrir el proyecto en VS 2022
- Seleccionar la plataforma deseada (Android, iOS, Windows)
- Ejecutar la aplicación
